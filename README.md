# Gestió de Cotxes

## Autor
 Enric Climent Martí
 

## Requeriments

Gradle, JDK >=7 


## Instalació i execució

```
gradle build
gradle eclipse
gradle compilar
java -jar build/libs/cotxes.jar

```
